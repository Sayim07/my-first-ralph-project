\# Todo App



A complete todo application built autonomously by \*\*Ralph Loop\*\* from a simple PRD.



\## Features

\- ✅ Add new tasks

\- ✅ Mark tasks complete (strikethrough)

\- ✅ Delete tasks

\- ✅ LocalStorage persistence (tasks survive page refresh)

\- ✅ Dark/Light theme toggle

\- ✅ Task counter (X tasks remaining)

\- ✅ Responsive design

\- ✅ Smooth animations



\## How to Use

1\. Open `index.html` in any web browser

2\. Type a task and click Add

3\. Click ✓ to mark complete

4\. Click ✗ to delete

5\. Toggle 🌙/☀️ for dark/light mode
